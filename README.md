# 📊 Excel Sales Dashboard

This project showcases a sales dashboard created using Microsoft Excel to analyze and visualize sales, profit, and category-wise performance.

## 🔹 Features
- Profit by Category
- Monthly Sales Trend
- Total Sales & Profit (KPI)

## 🛠 Tools Used
- Excel
- Pivot Tables
- Charts
- Data Cleaning

## 📁 Dataset
Superstore Sales Data

## 🚀 Outcome
Built an interactive dashboard for better business insights.
